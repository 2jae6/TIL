# XcodeGen 에서 Tuist 전환 프로젝트

1. yml 이 아니라 swift 라서 컴파일 시 오류를 더 잘 잡음
2. XcodeGen 도 멀티프로젝트 멀티타겟이 가능하지만 트위스트는 좀 더 쉽게 지원함
3. 클래스 다이어그램을 지원함 (물론 Tuist 안써도 가능함 SwiftPlantUML 혹은 )
